# SQL Mini Project: Patient Health Analysis

## Objective
To analyze patient health data and identify abnormal test results and trends using SQL.

## Dataset Overview
- 5 Patients
- 3 Lab Tests
- Multiple test results

## SQL Skills Used
- Joins
- CASE WHEN
- Aggregations (COUNT, AVG)
- GROUP BY
- Filtering

## Key Analysis Performed
- Identified normal vs abnormal test results
- Counted abnormal cases per test
- Analyzed results by age group
- Detected high cholesterol patients
  
## Key Insights
- High cholesterol observed in older patients
- Some patients show low hemoglobin levels
- Most glucose results fall within normal range

## Files Included
- schema.sql → Table creation
- data.sql → Data insertion
- analysis_queries.sql → All SQL queries

## Tools Used
- MySQL Workbench
- SQL
